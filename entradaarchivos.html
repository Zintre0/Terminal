Entrada de archivos

La entrada de archivos puede ser utilizada para subir archivos al servidor. Este control muestra una caja de texto donde el usuario debe especificar la ruta del archivo (que será adjuntado localmente por el navegador) que será enviado al servidor.

Nota que para los formularios con subida de archivos debes especificar el valor "multipart/form-data" en el atributo "enctype" del tag HTML form, de otro modo, el archivo no será enviado.



Código
<form method="post" action="agente.php" enctype="multipart/form-data">
Ingresa el archivo: <input name="imagen" type="file" />
</form>

Formularios para correos

También podemos crear formularios de contacto en nuestras páginas que enlacen directamente a nuestro cliente de correo modificando el campo "action"

Código
<form action="mailto:emaild@email.com" method="post" enctype="text/plain" >
<input type="hidden" name="opcion" value="si">
</form>
