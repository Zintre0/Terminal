Entrada textual

Entrada de línea

Este control recolecta información textual en una sola línea sin saltos. En la mayoría de los formularios, la tecla "enter" presionada en uno de estos campos, envía la información actual del campo.

Este control es insertado en documentos HTML usando el tag HTML input con el valor "text" en su atributo "type". El valor inicial, mostrado cuando la página se carga, puede ser definido usando el atributo "value".

Código
<form method="post" action="algo.php">
Ingresa un texto: <input name="ingresodetexto" type="text" value="Tu texto va aquí." />
</form>

Entrada de multi-línea

Este control permite a los usuarios ingresar texto en una o más líneas. Se inserta utilizando el tag HTML <textarea>. En este tag, el contenido será el texto inicial. Nota: los tributos "rows" y "cols" son las dimensiones de la caja de texto.

Código
<form method="post" action="algo.php">
Ingresa tus comentarios:<br /><textarea name="comentarios" rows="2" cols="30">...Esta ayudantía es fome...</textarea>
</form>

Entradas de password

Este control es una entrada de línea con la diferencia de que "esconde" los carácteres ingresados mostrándolos como puntos o asteriscos para evitar que los usuarios vean su contenido.

Es definido utilizando el valor "pass" en el atributo "type", y su valor inicial puede ser definido usando el atributo "value".

Código
<form method="post" action="algo.php">
Ingresa tu contraseña: <input name="contrasenia" type="password" value="123456" />
</form>
